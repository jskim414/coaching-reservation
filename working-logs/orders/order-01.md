# 작업 수정 요청

## 메인페이지 메뉴
 - 관리자 화면 메뉴 삭제
 - 관리자 페이지는 수동으로 /admin 입력시 접속할 수 있도록 수정

## 예약 신청 페이지

### 예약 신청 정보 입력 Form
 - 항목 추가
  - 환불시 입금받으실 계좌번호 (은행, 계좌번호, 예금주)
  - 예약 비밀번호 (예약 조회시 활용 목적)

## 예약 조회
  - 예약 조회 필터
    - 예약자 이름
    - 예약자 전화번호 OR 예약자 이메일
  - 입력 후 조회 결과 출력
  - 개별 예약 클릭해서 확인하려면 예약 비밀번호 입력

## 관리자 화면

### 예약 조회
- 편의를 위한 필터링 추가
  - 기간별 조회 필터 추가

### 예약 접수 목록
 - 일괄 편집할 수 있도록 각 목록 앞에 체크박스 추가
 - '삭제' 버튼 추가 (DB에서 삭제)

### 예약 상세
 - 삭제 버튼 추가 (삭제할 수 있게)
 
 
### 일정 관리
 - 시작 시각 입력 후 종료 시간 자동 계산할 수 있는 기능 제공 (예 : 프로그램 진행 시간(분) 기준으로 자동 계산. 이후 수동으로 변경 가능)
